# 更新日志

本项目遵循语义化版本，最新版本在最上方。

## v1.3.0

新增

- **删除历史**：面板顶部新增「🕘 历史」按钮。每次删除的完整备份都会存入本地数据库（IndexedDB），可随时查看、下载或还原。
  - 历史记录**不受「删除时下载备份」开关影响**——即使当场没下载文件，事后也能从历史里找回。
  - 每条记录支持：下载备份、一键还原、单条删除；顶部可「清空全部」。
  - 自动保留最近 60 批，超出丢弃最旧的。
  - 数据存在本浏览器本地，换设备或清理浏览器数据会丢失。

## v1.2.1

修复

- 取消勾选「删除时下载备份」后，删除角色卡仍会下载备份的问题（现在严格遵循开关）。
- 批量删除角色卡时触发浏览器「允许下载多个文件」拦截、被中断后静默失败的问题：现在一批备份**打包成单个文件**下载（有 JSZip 时打成 zip，内含可直接重新导入的 PNG；否则合并为单个 JSON），彻底避开多文件拦截。
- 角色卡无法导出的问题：改用官方 `/api/characters/export` 接口，导出的是完整可用的 PNG。

改进

- 备份失败时不再静默继续删除，而是**跳过该项并提示**，避免无备份误删。
- 角色卡现在也支持「撤销上次删除」与从历史还原（走官方导入接口）。

## v1.2.0

修复

- **手机端只显示半屏 / 面板背景透明** 的问题。根因是自建浮层的 `position: fixed` 被酒馆带 `transform` 的容器劫持。
- 改为使用 SillyTavern 内置 Popup 系统承载面板，手机酒馆下像原生弹窗一样全屏，深浅色主题都贴合。JSON 编辑器同样改用酒馆弹窗。

## v1.1.0

改进

- 首轮手机端适配：面板全屏化、标签横向滑动、操作按钮在触屏下常显、搜索框与按钮针对窄屏重排。

## v1.0.0

首个版本

- 一个面板集中管理五类数据：预设（OpenAI/TextGen/NovelAI/Kobold 及 Instruct/Context/SysPrompt/Reasoning）、世界书、角色卡、聊天记录、主题美化。
- 功能：实时搜索筛选、多选批量删除、单条改名、直接编辑 JSON、单条导出、删除前自动备份 + 一键撤销。
- 全部走 SillyTavern 官方后端接口，云端/服务器部署同样可用。
